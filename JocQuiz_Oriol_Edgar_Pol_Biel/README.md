# Rèptor - Joc Quiz Web

## Descripció
Projecte web d'un joc de preguntes tipus quiz creat amb HTML, CSS i JavaScript.

El jugador introdueix el seu nom, tria un mode de joc i respon 15 preguntes aleatòries. Al final es mostren els resultats i un rànquing local guardat al navegador.

## Funcionalitats

- Pantalla d'inici amb nom obligatori.
- Selecció de mode de joc.
- Mode Cultura.
- Mode Banderes.
- 15 preguntes aleatòries per partida.
- Temporitzador de 15 segons per pregunta.
- Validació visual de resposta correcta i incorrecta.
- Pantalla de càrrega entre preguntes.
- Pantalla final de resultats.
- Rànquing local amb LocalStorage.
- Disseny responsive.

1. Obrir la carpeta del projecte amb Visual Studio Code.
